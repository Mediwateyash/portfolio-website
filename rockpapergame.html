Rock papper scissor Game

Project Overview

The Rock-Paper-Scissors Game is a fun and simple interactive game developed using HTML, CSS, and JavaScript. It's a classic hand game usually played between two people, where each player simultaneously forms one of three shapes with their hand: rock, paper, or scissors. In this project, I have built a single-player version of the game where users can play against the computer.

Video Demonstration

Project Details

Technologies Used

  • Frontend: HTML, CSS, JavaScript

Features

  • Gameplay: The player chooses between Rock, Paper, or Scissors, while the computer randomly selects its choice.
  • Result Calculation: The winner (player or computer) is instantly displayed based on game rules.
  • Score Tracking: The game maintains a running score of both the player and the computer.
  • Responsive Design: The game layout adjusts smoothly across devices, ensuring compatibility with both desktop and mobile screens.

Purpose and Goals

  • Learning JavaScript: This project demonstrates my understanding of basic JavaScript functions, event handling, and game logic implementation.
  • User Interaction: The game enhances my ability to handle real-time user inputs and dynamically update the interface based on game results.
  • Responsive Design: Ensuring smooth gameplay on all devices was a major design goal for this project.

Future Enhancements

  • Add sound effects to enhance user experience during gameplay.
  • Implement a multiplayer mode for two players on the same device.
  • Introduce animations for better visual feedback.
  • Reset score feature after a specified number of rounds.
