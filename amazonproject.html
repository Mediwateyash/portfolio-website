Amazon website clone

Project Overview

The Amazon Clone is a front-end replica of the popular e-commerce website Amazon.com, created using HTML and CSS. This project aims to replicate the visual design and layout of Amazon's homepage, including key features such as navigation bars, search functionality, and product categories. The clone provides a realistic appearance of the original site while serving as a demonstration of front-end development skills.

Video Demonstration

Project Details

Technologies Used

  • Frontend: HTML, CSS

Features

  • Header: Amazon logo positioned on the top left. Search bar and dropdown menu with options like "Hello, Sign in," "Account," "List," "Return," "Orders," and "Cart."
  • Navigation Bar: Main navigation options including "All," "Today's Deals," "Customer Service," "Registry," "Gift Cards," "Sale," and "Shop by Category."
  • Hero Section: Hero section with advertisements similar to the original Amazon website.
  • Product Categories: Sections for various categories such as Clothes, Health and Personal Care, Furniture, Electronics, Makeup, Pet Care, New Arrivals, and Toys.
  • Footer: Contains Amazon logo, privacy information, and links to various sections like "Careers," "Blogs," "Make Money with Amazon," and "Disclaimer." Includes a "Back to Top" button for easy navigation.
  • Design Elements: Mimics the original site's appearance with features like a white border on hover for navigation elements.

Purpose and Goals

  • Demonstration: Showcasing the ability to replicate a complex, well-known website using HTML and CSS.
  • Visual Fidelity: Achieve a high degree of similarity to the original Amazon homepage, including layout, design, and key features.
  • Skill Showcase: Highlight proficiency in front-end development and attention to detail in mimicking a professional website.
