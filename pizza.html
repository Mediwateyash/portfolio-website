Online Pizza Delivery

Project Overview

Pizza Stop is an online pizza ordering platform that allows users to browse, select, and order pizzas from various categories. The website features both a user interface for customers to place and manage their orders and an admin panel for managing orders, menu items, and customer interactions. The platform aims to streamline the ordering process while providing a user-friendly experience.

Project Details

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP
  • Database: SQL

Features

  • User Panel:
    • Homepage Navigation: Users can explore different pizza categories, view menu items, and place orders.
    • Order Process: Users can add pizzas to their cart, customize their order, and choose between Cash on Delivery or online payment options.
    • Order Tracking: Users can track the status of their orders in real-time, from preparation to delivery.
    • Profile Management: Users can update personal information, view order history, and manage their account settings.
  • Admin Panel:
    • Dashboard Management: Admins can manage orders, update order statuses, and assign delivery personnel.
    • Menu and Category Management: Admins can add, edit, or delete pizza categories and menu items, including descriptions, prices, and images.
    • User Management: Admins can manage user roles, update user information, and view chat histories.
    • Site Settings: Admins can update site information, including contact details and address.

Purpose and Goals

  • Purpose: To provide a comprehensive platform for ordering pizzas online, making it easier for customers to place orders and for admins to manage those orders efficiently.
  • Goals:
    • To create a user-friendly website that allows customers to easily browse and order pizzas.
    • To implement an admin panel that streamlines order management and improves customer service.
    • To provide a platform that can be expanded with additional features in the future, such as online payment integration and more detailed tracking.
